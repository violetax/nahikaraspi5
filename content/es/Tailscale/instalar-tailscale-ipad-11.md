# Instalar Tailscale

## Requisitos

- iPad con iPadOS.
- Apple ID.
- Conexión a Internet.

## Instalar

Abrir **App Store**.

Buscar:

```
Tailscale
```

Instalar la aplicación oficial.

## Iniciar sesión

Abrir Tailscale.

Pulsar **Log in**.

Elegir:

- Google
- Microsoft
- GitHub
- Apple
- etc.

Aceptar la creación de la VPN cuando iPadOS lo solicite.

## Comprobar

El icono **VPN** aparecerá en la barra de estado cuando Tailscale esté conectado.

El iPad ya puede acceder a los demás equipos de la red Tailscale.
