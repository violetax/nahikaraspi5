# Instalar Tailscale

## Requisitos

- iPhone con iOS.
- Apple ID.
- Conexión a Internet.

## Instalar

Abrir **App Store**.

Buscar:

```
Tailscale
```

Instalar la aplicación oficial.

## Iniciar sesión

Abrir Tailscale.

Pulsar **Log in**.

Elegir:

- Google
- Microsoft
- GitHub
- Apple
- etc.

Aceptar la creación de la VPN cuando iOS lo solicite.

## Comprobar

El icono **VPN** aparecerá en la barra de estado cuando Tailscale esté conectado.

El iPhone ya puede acceder a todos los dispositivos de la red Tailscale.
