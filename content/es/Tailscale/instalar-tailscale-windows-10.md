# Instalar Tailscale

## Requisitos

- Windows 10.
- Conexión a Internet.

## Descargar

Abrir:

https://tailscale.com/download

Descargar la versión para Windows.

## Instalar

Ejecutar el instalador.

Aceptar las opciones por defecto.

## Iniciar sesión

Abrir **Tailscale** desde el menú Inicio.

Pulsar **Log in**.

Elegir una cuenta:

- Google
- Microsoft
- GitHub
- Apple
- etc.

## Comprobar

El icono de Tailscale debe aparecer en la bandeja del sistema.

El equipo ya forma parte de la red privada Tailscale.
