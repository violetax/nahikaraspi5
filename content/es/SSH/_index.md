# QUÉ ES

> **SSH (Secure Shell)** es un protocolo.**

> Un protocolo es un **sistema de normas** que regulan la comunicación entre dos o más sistemas. 

> Permite CONECTARSE DE FORMA SEGURA A OTRO ORDENADOR a través de una red para administrarlo como si estuvieras delante de él.

Con **SSH** puedes:

- iniciar sesión en otro equipo;
- ejecutar comandos;
- copiar archivos;
- administrar el sistema de forma remota.

SSH cifra toda la comunicación entre ambos equipos, de modo que nadie puede leer ni modificar los datos durante la conexión.

En esta documentación, SSH se utilizará para acceder a la Raspberry Pi desde un ordenador, una tablet o un teléfono móvil.
