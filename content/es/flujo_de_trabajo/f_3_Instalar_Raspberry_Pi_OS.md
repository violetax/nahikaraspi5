# 3. Instalar Raspberry Pi OS

Utilizar **Raspberry Pi Imager**.

En **Edit Settings** configurar, al menos:

- hostname
- usuario
- contraseña
- SSH
- autenticación mediante clave pública
- clave pública generada en Windows
- Wi-Fi (si procede)
- zona horaria
- distribución del teclado.
